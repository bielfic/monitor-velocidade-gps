Sugestões para resolver problemas de localização:

  • Atenção no Safari/iOS (iPhone): O iOS exige conexão **HTTPS** (cadeado seguro) para liberar o GPS. Se você acessar via `http://` com o IP local, o iPhone nega o acesso automaticamente sem perguntar.
  • Certifique-se de que o GPS está ativado nas Configurações do iOS (Privacidade e Segurança > Serviços de Localização > Sites do Safari).
  • Verifique se a opção "Localização Precisa" está ativada no iOS.
  • Tente abrir o site via HTTPS ou implantado na nuvem (ex: Vercel, Render, GitHub Pages).
  • Se estiver usando o Brave ou Safari, verifique os bloqueadores de privacidade.
