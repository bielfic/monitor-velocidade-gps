Sugestões para resolver problemas de localização:

  • Certifique-se de que o GPS está ativado no dispositivo.
  • Verifique se o navegador tem permissão para acessar a localização.
  • Tente usar o aplicativo em um local aberto com boa visibilidade para o céu.
  • Se estiver usando o Brave, verifique as configurações de privacidade e habilite a geolocalização.
  • Atualize o navegador para a versão mais recente.
